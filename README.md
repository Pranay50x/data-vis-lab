Part A: Data Visualization with Python
1. Write a program to perform time series visualization using Pandas and Matplotlib,
showing trends, seasonal patterns, and moving averages.
Given the "Airline Passengers" dataset (1949–1960), write a Python program to:
- Pre-process the data by converting date strings into a DatetimeIndex.
- Calculate a 12-month Rolling Mean to smooth out noise and visualize the growth trend.
- Generate a Seasonal Boxplot to identify recurring monthly fluctuations (Seasonality).

2. Write a program to create pair plots and regression plots using Seaborn, to study
variate relationships and trends.

3. Write a program to create interactive visualizations using Plotly, including interactive line, bar, and scatter plots with hover information.

4. Write a program to design an interactive dashboard using Plotly, incorporating multiple charts, filters, and dropdown controls.

5. Write a program to visualize geographical data, using map-based plots to represent location-based information.

6. Write a program to visualize machine learning model outputs, including confusion matrices, ROC curves, and feature importance plots.
